Release notes

New in version 9.0 of Camera Verifier 15000 series (26 November 2025)

  • Add “multiple codes” PDF & HTML reports.
  • Add “Japanese convenience store” plugin.
  • Improve Data Matrix grid accuracy for small codes.
  • Check for missing drivers at startup.
  • Update Dutch translation.
  • Various user data improvements.
  • Fix linear code modulation visualisations to cover both elements of each edge.
  • Fix spurious addons found next to EAN-8 codes.
  • Various stability fixes.

New in version 8.1 of Camera Verifier 15000 series (28 July 2025)

  • Improve decoding adjacent codes with overlapping quiet zones.
  • Improve consistency of GS1 DataBar decodability parameters.
  • Improve linear code per-element & per-character visualisations.
  • Fix invalid window splitter positions at startup.
  • Fix recommended aperture for larger Performance Qualification codes.
  • Fix some instances of Laetus codes misdecoding as MSI Plessey.
  • Fix reference decode of MSI Plessey codes.

New in version 8.0 of Camera Verifier 15000 series (22 April 2025)

  • Implement changes for ISO/IEC 15415:2024: continuous grading, revised threshold and reflectance, print growth, combination of modulation and reflectance margin.
  • Implement changes for ISO/IEC 15416:2025: revised threshold and reflectance.
  • DPM updates for ISO/IEC 29158:2025.
  • Data Matrix updates for ISO/IEC 16022:2024.
  • QR updates for ISO/IEC 18004:2024.
  • Add an option to export the audit log as a PDF.
  • Add a login session timeout option.
  • Update GS1 plugins for GS1 General Specifications v25.0.
  • Add check for deprecated YYMM00 date format in regulated healthcare plugins.
  • Report Laetus standard & inverted decodes separately.
  • Make decode direction of vertical Laetus codes consistent with specification.
  • Update French & Polish translations.
  • Fix user data dialogue popping up too many times when auto saving or printing.
  • Assorted bug fixes & stability improvements.

New in version 7.1 of Camera Verifier 15000 series (20 September 2024)

  • Fix decoding of some rectangular Data Matrix codes.
  • Fix finding of some Data Matrix codes when a linear code is present in the image.
  • Fix the clearing of user data field content after entering and leaving the settings dialogue.
  • Add support for Laetus box code.
  • Improve bounding boxes for linear components of composite codes.
  • Fix decoded messages of linked codes.
  • Improve DPM enable toolbar icon.
  • Various stability fixes.
  • Update Chinese (Traditional) translation.

New in version 7.0 of Camera Verifier 15000 series (02 July 2024)

  • Add templates for auto save/report file and directory name generation.
  • Add option to filter by date range when exporting the audit log.
  • Add option to disable automatically checking for updates.
  • Full implementation of the dot connecting algorithm when in DPM mode and a “Dot connecting” parameter now shows whether it was used to obtain the final grade.
  • Validate GS1 Digital Link data content in the “GS1 Retail 2D” plugin.
  • Improve error displayed by any incompatible plugins shown while in DPM mode.
  • Improve decoding of poor EAN/UPC codes, including addons.
  • Clarify wording of DPM controls in Setup dialogue.
  • Don’t allow editing the append report file name when appending is not enabled.
  • Only enable UserData import if the currently loaded scan contains user data items.
  • Show the function 1 character at the start of the GS1 DataMatrix decoded message.
  • Update Danish, Dutch, French, Italian, Polish, German & Russian translations.

New in version 6.0 of Camera Verifier 15000 series (02 February 2024)

  • Add “GS1 Retail 2D” & “HDA shipping case” plugins.
  • CIP plugin is now deprecated, and will be removed in a future update.
  • Update GS1 plugins as per GS1 General Specifications v24.0.
  • Update Dutch translation.
  • Add colour for status to more parameters, e.g. linear quiet zones.
  • When a code has been found, clicking on the code will stop searching and verify the code.
  • Improve zoom when clicking on verified bar code.
  • Improve bounding box detection for PDF417 & composite components.
  • DPM analysis now performed automatically with both a 50% and an 80% aperture.
  • Matrix symbols that have been marked with disconnected dots are now more readily found and decoded in DPM analysis mode.
  • Fix Code 39 check character plugin failing PZN codes.
  • Fix various linear decoding issues.

New in version 5.0 of Camera Verifier 15000 series (06 July 2023)

  • DPM analysis of QR codes to ISO/IEC 29158.
  • Show enhanced exposure when scanning in DPM mode.
  • DPM mode can now be toggled from the toolbar, if enabled in settings.
  • Kanji mode for QR implemented, including new Shift JIS plugin.
  • Add “PZN – Code 39” plugin.
  • Right-click menu to copy plugin output to clipboard.
  • Show Extended Channel Interpretation (ECI) in encodation details.
  • Update Danish, Dutch, French, German, Italian, Portuguese, Russian & Spanish translations.
  • Fix ‘ghost’ decodes of previous codes when double-clicking ‘Decode’ button.
  • Fix some instances of Laetus codes misdecoding as MSI Plessey.
  • Reduce the incidence of phantom Laetus codes being found in text & graphics.
  • Correct overly pessimistic overall grades for EAN/UPC codes with addons.
  • Fix possible optimistic defects grades in linear codes with low contrast.
  • Fix linear threshold parameter not displaying the global threshold used by the reference decode algorithm if it fails.
