About Logistic System

LS is a web-based solution that provides wireless tracking and mobile communication for managing vehicles and field personnel. The application includes the following sections:

  • Managing Field Personnels
  • Managing Vehicles
  • Managing Jobs
  • Web based Management Reports



The service leverages the latest satellite navigation and mobile technology to offer real-time visibility of parcels, vehicles, and personnel.

LS is a fully hosted cloud computing solution that provides services at a fraction of the cost of an in-house system. With cloud computing, you can access your system and business data 24/7 from anywhere with an Internet connection. This secure, efficient, and powerful service continuously offers the computing resources your business needs. LS ensures that your data, which is central to your business's success, is always secure and backed up.

LS offers a comprehensive suite of applications, including both office-based and field-based modules. LS Office operates on a standard PC with a web browser and internet connection, while LS Mobile runs on a PDA with a wireless internet connection.
A unique feature of this system is its ability to capture and integrate GPS data from vehicles and field personnel activities in real time, providing clear visibility of resource activities at any given moment.
Web-based "Management Reports" have been developed using tracking information from the tracking device and data from the PDA. These reports detail various activities of vehicles and field personnel.

This application is designed for logistics, distribution, and paper/magazine delivery companies, offering real-time visibility of the entire operation using the latest technologies. Various management reports are available and can be customised to meet user requirements.

Logistic System Office

LS Office provides your business with a total solution from call logging through to completion of delivery and collection. The Call Centre staff will have visibility of parcels, vehicles and personnel in real-time. LS Office enables Call Centre staff to log call details against predefined customer details. Once a call is logged, LS utilises advanced resource management to allocate the best-fit vehicle and field personnel. Call details are then communicated to the field personnel via PDA or phone. At the point of delivery a signature is captured.

LS consists of a web based user-friendly Operational Panel for the operational team to manage the total logistics operation. The operational panel is categorised into managing field personnel, vehicles, jobs and customer information. Where the operational team will be able to Add, Edit and Update various tasks in order to manage the logistic operation.

LS consists of a single screen (LOS) where the unique feature of this system is that the GPS data from the vehicle and activity of the field personnel are captured and integrated in real time which provides a clear visibility of the resources activity at any given instance. And the mobile personnel activity can be viewed on gantt chart in real-time. All vehicles and mobile personnel activity can be viewed on map. LS Office gives you the foundations for a professional, cost effective logistics operation.

Logistic System Mobile

LS Mobile is a pioneering PDA based solution to help you manage your field based logistic operations. Its based on satellite navigation and mobile technology to give you visibility of field operations in real time through LS Office. The PDA uses a wireless internet connection to connect to LS office.

LS Mobile enables the field personnel to log call against predefined customer details. Call details are communicated from LS Office via the LS data centre to field personnel PDAs. Once a field personnel logs on, they can view there deliveries and collections in the jobs folder and can capture signatures on collection and delivery.

The PDA communicates with the LS data centre at predefined intervals and updates LS Office at the end of every activity.

Driver Activity

Activity

This report generates the comprehensive activity of mobile personnel for a particular date and time range, which will enable the user to view and thoroughly analyse the tasks the mobile personnel carried out during a particular period.

Idle Time

This report generates the idle time a mobile personnel for a particular date and time range, which will enable the user to view the time durations when the mobile personnel is not involved in carrying out in any tasks


Sign In/Out

This report generates the Sign In/Out of the mobile personnel for a particular date range, which will enable the user to view the start and end time and the hours worked during a particular period

Part not Found

This report generates the number of parts not found by each mobile personnel for a particular date range, which will enable the user to view the parts which could not be collected due to part missing on site.

Vehicle Activity Report

Activity

This web based report generates the activity of a vehicle for a particular date and time range, which will enable the user to view and analyse the vehicle usage for a particular period.

Stop Vehicle

This report generates the duration between vehicle Ignition on and off, which will enable the user to view the time durations where vehicle not being used.

Total Mileage

This report generates the total miles driven by a vehicle for a particular date and time range, which will enable the user to view the vehicle utilisation

Idle Time

This report generates the idle time for a vehicle for a particular date and time range, which will enable the user to view the time durations when the vehicle ignition is on but the vehicle is idle and not moving

Management Reports

The management reports consist of five categories, with details as below

Summary Report

This report generates the number of parts and time taken for collection and delivery of the parts for a particular period, which will enable the user to view and analyse the part usage trend and the time trend for all the activities

Compare by Parts Report

This report is for comparing the usage of various parts against a particular device, which will help the customer to review the stock levels to meet the demand



Compare by Vehicles Report

This report is for comparing the number of jobs carried out by a particular vehicle in comparison with other vehicles, which will enable the customer to plan and reallocate the area to assign equal number of jobs to all vehicles

Compare by Mobile Personnel Report

This report is for comparing the number of jobs carried out by a particular mobile personnel in comparison with other personnel, which will enable the customer to measure the KPIs of the mobile personnel in comparison with other personnel

FRC Job Allocation

This report is for analysing how the mobile personnel being utilised by the control centre staff during their shifts, which will enable the customer to identify the control centre staff who need further help towards better usage of field personnel.
